# Janus Food Group QMS

Quality Management System for Tetra Recart production — 21 CFR Part 11 compliant batch tracking, electronic signatures, and workflow management.

## Features

- **Batch Tracking**: In-Process → Filling → Post-Retort → Incubation → Release
- **Electronic Signatures**: CFR 21 Part 11 compliant with signature meanings
- **OOS Detection**: Real-time out-of-specification alerts with supervisor approval workflows
- **Product Setup Wizard**: Configure specs, samples, and certifications
- **Audit Trail**: Complete traceability for all actions

## Demo

Open `index.html` in any browser. Demo accounts are available on the login screen.

## Tech

Self-contained HTML/CSS/JS — no build step, no dependencies.
